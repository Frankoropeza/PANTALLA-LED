## Pantalla LED Interior

Las pantallas LED para interior ofrecen **imágenes nítidas y colores vibrantes** a cortas distancias de visión. Perfectas para ambientes controlados donde se requiere impacto visual de alta calidad.

### Características principales

- **Alta resolución**: Pixel pitch fino (P2 a P4) para imágenes ultra nítidas
- **Brillo controlado**: 800 a 1,500 nits ajustables automáticamente
- **Diseño ultradelgado**: Gabinetes compactos que se integran a cualquier interior
- **Instalación versátil**: Pared, techo, pedestal o estructura personalizada
- **Silenciosa**: Sin ventilación activa, operación silenciosa

### Aplicaciones

- Lobbies corporativos y de hoteles
- Centros comerciales y tiendas departamentales
- Restaurantes y bares (menús digitales)
- Salas de conferencias y auditorios
- Museos y exhibiciones interactivas
- Hospitales y clínicas (señalización digital)
- Casinos y centros de entretenimiento

### Especificaciones técnicas

| Parámetro | Valor |
|-----------|-------|
| Pixel Pitch | P2 / P2.5 / P3 / P4 |
| Luminosidad | 800 – 1,500 nits |
| Ángulo de visión | 160° H / 140° V |
| Frecuencia de refresco | ≥ 3,840 Hz |
| Temperatura de operación | 0°C a +45°C |
| Consumo promedio | 150 W/m² |
| Voltaje | 110V / 220V AC |
| Vida útil | ≥ 100,000 horas |

### Control de contenido

Incluye sistema de gestión de contenido (CMS) para programar y transmitir contenido de forma remota desde cualquier dispositivo. Compatible con señal HDMI, DVI y transmisión por red.
